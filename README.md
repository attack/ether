# Ether

[![Gem Version](https://badge.fury.io/rb/ether.png)](http://badge.fury.io/rb/ether)

Early stages of a lightweight API consuming framework.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ether'
```

Or install it yourself as:

```shell
$ gem install ether
```
